# Va2: Crawl
Website SEO crawler and debugger.

# Installation & Usage
Just download `install.sh` and run `sh install.sh`, then call the script using:
```
lua crawl.lua
```
You can also set alias in your `~/.bashrc`, for example:
```
alias crawl='lua ~/Documents/lua/crawl.lua'
```
